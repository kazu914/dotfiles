# My dotfiles repo
This repo includes dotfiles for
- `zsh`
- `nvim`
- `git`

# How to install

```zsh
# Clone this repository
git clone https://github.com/kazu914/dotfiles

# Move to the repository
cd dotfiles

# Initialize some additional packages
make

source ~/.zshrc
```
